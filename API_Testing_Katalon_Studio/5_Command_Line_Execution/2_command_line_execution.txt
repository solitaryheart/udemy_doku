************* GENERATING A CMD-COMMAND ******************
*********************************************************


1.) Open katalon Studio
2.) Select a Test Suite for Execution 
3.) Click on Button "Build CMD"
4.) Select a Test Suite
5.) For CI-Integrations - "Other Options" should be unchecked
6.) Click on "Generate Command"

----------------------------------------
katalon -noSplash  -runMode=console -projectPath="C:\Users\matthias.meier\Katalon Studio\API_Testing_Udemy_KatalonStudio\API_Testing_Udemy_KatalonStudio.prj" -retry=0 -testSuitePath="Test Suites/KatalonAnalytics-API" -executionProfile="default" -browserType="Web Service"

------------------------------------------------------

7.) switch to folder where katalon studio is installed

-> C:\Katalon_Studio\Katalon_Studio_Windows_64-6.1.3\Katalon_Studio_Windows_64-6.1.3

9.) paste command and execute

--------------------------------------

************* GENERATING A PROPERTY FILE ******************
************************************************************

1.) Open katalon Studio
2.) Select a Test Suite for Execution 
3.) Click on Button "Build CMD"
4.) Click on "Generate Property File" button -> console.properties
5.) Save File, where Project is installed

#Tue Jun 25 08:42:51 CEST 2019
browserType=Web Service
executionProfile=default
projectPath=C\:\\Users\\matthias.meier\\Katalon Studio\\API_Testing_Udemy_KatalonStudio\\API_Testing_Udemy_KatalonStudio.prj
retry=0
testSuitePath=Test Suites/KatalonAnalytics-API

-------------------------------------

6.) File can be used for cmd execution by providing the file name
7.) Go to location where katalon studio is installed (cmd) and provide location where properties file is installed
   (project location)

katalon -propertiesFile="C:\Users\matthias.meier\Katalon Studio\API_Testing_Udemy_KatalonStudio\console.properties" -runMode=console
