NODES

- three types of nodes

a) Element/Tag
b) Attribute
c) Text

<tagName attribute='attributeValue'>Text</tagName>

topmost element of tree is called root element


ATOMIC VALUES

- are nodes with no children or parent

ITEMS

- are atomic values or nodes


RELATIONSHIP OF NODES

a) parent - each element and attribute has one parent - except root
b) children - element bodes may have ZERO, ONE, or MORE children
c) siblings - nodes have the same parent (Geschwister)
d) ancestors - a nodes parent, parent's parent etc (Vorfahren)
e) descendants - a node's children, children's children etc. (Nachkommen)
